# Hexagonal architecture with Nest.js and CQRS


Example project building a Gym admin system.


## run app

```bash
# dev
$ npm run start

# dev-watch
$ npm run start:dev

# production
$ npm run build & npm run start:prod
```

## Testing

```bash
# unit testing
$ npm run test:unit

# e2e
$ npm run test:e2e

# all the tests
$ npm run test

# test coverage
$ npm run test:cov
```
